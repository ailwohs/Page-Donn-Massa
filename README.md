# 🍰 Donna Massa – Doces e Salgados

Site institucional desenvolvido para a confeitaria **Donna Massa – Doces e Salgados**, com foco em apresentação de produtos, identidade visual elegante e contato direto via WhatsApp.

---

## 🌐 Sobre o Projeto

Este projeto é uma **Landing Page institucional**, funcionando como vitrine digital da loja.

O site não possui sistema de compras online.  
Os pedidos são realizados exclusivamente via **WhatsApp**.

---

## 🎯 Objetivo

- Apresentar os produtos da confeitaria
- Exibir categorias como Bolos, Salgados e Doces
- Mostrar o cardápio da loja
- Divulgar redes sociais
- Facilitar o contato direto com botão de WhatsApp
- Manter identidade visual elegante e aconchegante
